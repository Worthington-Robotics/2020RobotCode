Detailed Description

This code runs all of the robot's loops. Loop objects are stored in a List object. They are started when the robot powers up and stopped after the match.

if (running_) {

loop.onLoop(now);
}
dt_ = now - timestamp_;
timestamp_ = now;
}
}
