\doxysubsection{Detailed Description}
This code runs all of the robot\textquotesingle{}s loops. \mbox{\hyperlink{interfacefrc_1_1lib_1_1loops_1_1Loop}{Loop}} objects are stored in a List object. They are started when the robot powers up and stopped after the match.
