\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classfrc_1_1lib_1_1control_1_1Path}{Path}} is a recording of the path that the robot takes. \mbox{\hyperlink{classfrc_1_1lib_1_1control_1_1Path}{Path}} objects consist of a List of Waypoints that the robot passes by. Using multiple Waypoints in a \mbox{\hyperlink{classfrc_1_1lib_1_1control_1_1Path}{Path}} object and the robot\textquotesingle{}s current speed, the code can extrapolate future Waypoints and predict the robot\textquotesingle{}s motion. It can also dictate the robot\textquotesingle{}s motion along the set path.
