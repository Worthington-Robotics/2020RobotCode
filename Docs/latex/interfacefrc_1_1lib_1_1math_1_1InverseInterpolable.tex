\doxysubsection{Detailed Description}
\mbox{\hyperlink{interfacefrc_1_1lib_1_1math_1_1InverseInterpolable}{Inverse\+Interpolable}} is an interface used by an Interpolating Tree as the Key type. Given two endpoint keys and a third query key, an \mbox{\hyperlink{interfacefrc_1_1lib_1_1math_1_1InverseInterpolable}{Inverse\+Interpolable}} object can calculate the interpolation parameter of the query key on the interval \mbox{[}0, 1\mbox{]}.
