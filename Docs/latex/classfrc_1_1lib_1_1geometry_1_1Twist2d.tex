\doxysubsection{Detailed Description}
A movement along an arc at constant curvature and velocity. We can use ideas from \char`\"{}differential calculus\char`\"{} to create new Rigid\+Transform2d\textquotesingle{}s from a \mbox{\hyperlink{classfrc_1_1lib_1_1geometry_1_1Twist2d}{Twist2d}} and visa versa. 

A Twist can be used to represent a difference between two poses, a velocity, an acceleration, etc.
