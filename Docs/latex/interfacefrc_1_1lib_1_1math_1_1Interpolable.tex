\doxysubsection{Detailed Description}
\mbox{\hyperlink{interfacefrc_1_1lib_1_1math_1_1Interpolable}{Interpolable}} is an interface used by an Interpolating Tree as the Value type. Given two end points and an interpolation parameter on \mbox{[}0, 1\mbox{]}, it calculates a new \mbox{\hyperlink{interfacefrc_1_1lib_1_1math_1_1Interpolable}{Interpolable}} representing the interpolated value.
